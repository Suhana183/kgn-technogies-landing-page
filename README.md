# 🚀 KGN Technologies - Responsive Landing Page

## 📌 Project Overview

KGN Technologies is a modern, responsive landing page designed for a fictional technology startup. The website showcases the company's services, key features, customer testimonials, and contact information using a clean and professional user interface.

This project was developed using **HTML5** and **CSS3** with responsive design principles, making it suitable for desktops, tablets, and mobile devices.

---

## 🌐 Live Demo

**Netlify:**
https://kgntechnogies.netlify.app/

## ✨ Features

* Responsive design for desktop, tablet, and mobile devices
* Modern and attractive user interface
* Navigation bar with logo
* Hero section with call-to-action button
* Feature highlights section
* Customer testimonials section
* Professional footer
* Google Search Console verification
* SEO-friendly meta tags
* Clean and well-organized code

```bash

```

### 2. Open the Project

Open the project folder in **Visual Studio Code**.

### 3. Run the Website

Open `index.html` directly in your browser or use the **Live Server** extension in Visual Studio Code.
